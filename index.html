React Mise

The React Store that you will enjoy using

Get Started
Demo

💡 Intuitive

Stores are as familiar as components. API designed to let you write well organized stores.

🔑 Type Safe

Types are inferred, which means stores provide you with autocompletion even in JavaScript!

🔌 Extensible

React to store changes to extend React Mise with transactions, local storage synchronization, etc.

🏗 Modular by design

Build multiple stores and let your bundler code split them automatically.

📦 Extremely light

React Mise weighs around 1kb, you will forget it's even there!
